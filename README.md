## Real-Time Device Tracking App

This project is a real-time device tracking application built with Node.js, Socket.io, and Leaflet. The application allows you to track and display the location of devices on a map in real-time.

## Features

Real-Time Tracking: View device locations as they update in real-time.
Interactive Map: Use Leaflet to render an interactive map for tracking device locations.
WebSocket Communication: Leverage Socket.io for real-time communication between the server and clients.

## Installation

Install Dependencies

```bash
npm install
```

Start the server

```bash
npx nodemon app.js
```
